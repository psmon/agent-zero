# AgentZero 홈 페이지 일러스트 — Gemini 이미지 프롬프트 모음

이 문서의 프롬프트들을 **Gemini (gemini.google.com 또는 Google ImageFX)** 에 그대로
복사해서 붙여넣으면 `home/index.html`이 기다리고 있는 8개 일러스트를 생성할 수
있습니다. 생성된 PNG는 `IMG/illustrations/` 폴더에 표시된 파일명으로 저장하면
홈 페이지에 자동 반영됩니다 (없으면 기존 스크린샷으로 자동 폴백).

---

## 공통 스타일 가이드 (모든 프롬프트의 윗줄에 자동 포함됨)

- **테마**: 다크 모드. 배경은 깊은 남색 (#0d1117) 또는 진한 보라/네이비 그라데이션
- **분위기**: 귀여운 (cute, kawaii-leaning), 둥글고 친근한 형태. 윈도우 창 같은 직사각형 UI 모양은 피한다
- **컬러 팔레트**: 시안(#56d4dd), 청록(#3fb950), 핑크(#ff79c6), 보라(#c586c0)를 강조색으로 사용
- **선**: 부드러운 둥근 모서리, 굵기 있는 외곽선
- **종횡비**: 가로형 1.6:1 (일러스트), 정사각형 1:1 (마스코트)
- **금지**: 사실적 사진, 실제 윈도우 OS UI 캡처, 너무 복잡한 디테일, 텍스트(영문/한글 모두 최소화)

## 저장 후 확인

1. 위 프롬프트로 PNG 생성
2. `IMG/illustrations/{파일명}.png` 로 저장
3. 커밋 → 푸시
4. GitHub Pages가 빌드되면 `home/index.html`이 자동으로 새 일러스트를 사용
5. 일부 파일이 빠져 있어도 페이지는 정상 동작 (해당 자리에는 기존 스크린샷이 폴백으로 표시됨)
