여러 작업 공간, 여러 터미널

여러 프로젝트와 여러 CLI 세션을 한 화면에서 나란히 다룰 수 있어요.

  • 작업 공간 그룹 — 사이드바에서 폴더를 별도의 작업 공간으로 묶을 수 있어요
  • 멀티탭 터미널 — 작업 공간마다 Claude Code, PowerShell, CMD, WSL, Codex, Git Bash를 자유롭게
  • ConPTY 렌더러 — Windows Terminal 엔진 기반, 색깔·유니코드·커서 모두 정상 동작
  • 지연 초기화 — 클릭한 탭만 실제로 터미널 프로세스를 띄워서 가벼워요

AgentBot — 채팅 중계와 AI 비서

작은 채팅 창 하나로 터미널 입력·키 전달·AI 채팅을 모두 처리해요. 3가지 모드를 한 창에서.

  • CHT — 활성 터미널에 글자를 그대로 보내요
  • KEY — 화살표·Esc·Tab·Ctrl 조합 같은 특수 키를 전달해요
  • AI — Gemma·OpenAI 같은 LLM과 SSE 스트리밍으로 채팅
  • 승인 감지 — Claude가 승인을 물어보면 자동으로 토스트를 띄워줘요
  • 스킬 동기화 — Claude Code의 /skills 결과를 자동으로 가져와 슬래시 명령으로 만들어요
  • MD 첨부 — 마크다운 파일을 최대 3개까지 AI 질의에 함께 보낼 수 있어요

CLI 자동화와 멀티 AI 연결

터미널을 코드로 제어하고, 여러 AI 에이전트를 터미널 입출력으로 이어 붙일 수 있어요.

  • terminal-list — 활성 세션 모두 찾기
  • terminal-send — 어느 탭에든 명령 보내기
  • terminal-read — 출력 캡처 (ANSI 코드 정리됨)
  • bot-chat — AgentBot에 메시지 푸시
  • AI ↔ AI 대화 — 서로 다른 AI CLI 사이에 메시지 라우팅
  • 크로스 모델 — Claude ↔ Codex, Claude ↔ Gemma 가능
  • TUI 추상화 — terminal-key가 CLI 별 입력 차이를 자동으로 맞춰줘요

작업 공간 노트

다른 도구를 띄우지 않고도 프로젝트 파일을 둘러보고 읽을 수 있어요.

  • 파일 트리 — .gitignore를 존중하는 폴더 탐색기
  • 문서 뷰어 — 마크다운, 30+개 언어의 코드, 이미지, PDF 모두 보임
  • 클립보드 기록 — 최근에 복사한 것들을 한 번에 다시 쓸 수 있어요

설정과 윈도우 스파이

터미널·LLM 공급자를 설정하고, 다른 윈도우의 글자를 캡처할 수 있어요.

  • CLI 정의 — CMD, PowerShell, Claude, WSL 등 어떤 터미널이든 등록 가능
  • LLM 설정 — 공급자 선택(Webnori/Gemma, LM Studio, OpenAI), 모델 선택, 즉석 채팅 테스트
  • AgentScrap — 십자선 드래그 캡처, 글자 추출, 요소 트리, 스크린샷

하네스 모니터와 AI 어시스트

하네스 설정을 보기 좋게 펼쳐 보고, 내장 AI로 직접 분석할 수 있어요.

  • 적응형 스키마 — v1/v2 하네스 설정을 자동으로 인식해 모든 섹션을 보여줘요
  • 펜슬 뷰어 — .pen 디자인 파일을 아키텍처 프레임으로 탐색할 수 있어요
  • AI 어시스트 패널 — 4가지 프리셋 질의 + LLM SSE 스트리밍 분석
  • 스킬 매크로 — AgentBot에서 하네스 스킬을 동기화하고 실행

데스크톱 자동화

마우스 클릭, 키보드 입력, 글자 캡처, 스크린샷, UI 요소 검사 — Claude Code에서 CLI 스킬로 모두 호출 가능.

IPC 파이프라인

WM_COPYDATA + 메모리 매핑 파일로 CLI↔GUI 통신. 보내고 잊거나 응답을 폴링할 수 있어요.

SQLite 영구 저장

윈도우 상태, 작업 공간 그룹, 터미널 탭, 클립보드 기록, LLM 설정 — 모두 저장하고 시작 시 복원.

최근 업데이트

개발 중에 새로 들어온 기능들을 사용자 관점에서 정리했어요.

신규 OS 제어 활성화 안내 — Claude가 혼자 윈도우를 조종
온보딩 화면에 [Register PATH] 한 번 누르면 Claude Code가 마우스·키보드·스크린샷 같은 AgentZero 기능을 AgentBot 창 없이도 단독으로 호출할 수 있다는 안내 섹션이 추가됐어요.
신규 agent-zero 스킬 — 카카오톡 채팅 자동화
Claude가 agent-zero 스킬로 카카오톡 창을 열고 채팅을 자동화할 수 있어요. 반복되는 메시지 발송이나 채팅 확인 같은 단순 작업을 Claude가 대신 해 줍니다.
신규 처음 사용자를 위한 온보딩 화면
앱을 처음 켜거나 새 버전을 설치하면 자동으로 안내 화면이 떠요. 자동 설치 체크리스트, Claude 첫 로그인 4단계, AgentBot 활용 팁이 한국어로 정리되어 있고 영어로도 볼 수 있어요.
신규 기본 CLI 목록에 Claude 자동 추가
설정의 CLI 정의 목록에 Claude가 기본으로 들어가 있어서, [+ CLI] 탭에서 클릭 한 번이면 Claude 터미널이 바로 떠요.
신규 하네스 모니터 — kakashi-harness v3 스키마 지원
하네스 모니터가 새 v3 스키마를 자동으로 인식해서 모든 섹션을 보여줘요. 펜슬(.pen) 디자인 파일도 화면을 꽉 채워 보여주니, 아키텍처 다이어그램을 더 크게 한눈에 볼 수 있어요.
개선 다크 테마 통일 — 스크롤바까지
모든 화면의 색감이 한 톤으로 정리됐어요. 그동안 살짝 튀던 스크롤바도 다크 테마에 맞게 바뀌어서 눈에 거슬리지 않아요.
